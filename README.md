# Real-Estate Platform

### Start to IOS

#### yarn build

```bash
yarn install
```

#### npm build

```bash
npm install
```

#### ios pod install

```bash
cd ios && pod update && cd ..
```

#### yarn run ios

```bash
yarn ios 
```

#### npm run ios

```bash
npm ios
```
